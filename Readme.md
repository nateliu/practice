## Practice

> Some fundamental technologies of the Rust language. This is from the Book "The Rust Programming Language".

## Quick Start
Un-comment the file function to run

``` bash
# Get the book
rustup doc
# Run With Cargo
cargo run
# Build for dev
cargo build
# Build for production
cargo build --release
```
